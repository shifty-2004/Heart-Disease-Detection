# ❤️ Heart Disease Detection

## 📌 Objective
Predict heart disease using Machine Learning.

## 📊 Dataset
Kaggle Heart Disease Dataset

## ⚙️ Tools Used
- Python
- Pandas, NumPy
- Scikit-learn

## 🤖 Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## 📈 Result
Random Forest gave best accuracy.

## 📌 Conclusion
Machine learning helps in early detection of heart disease.
